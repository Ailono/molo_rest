# Реализация: Оферта и согласие на обработку персональных данных в корзине

## Обзор

Реализация состоит из клиентской части (добавление чекбоксов согласий в форму заказа в cart.js), серверной части (валидация на /api/orders в server.js) и миграции базы данных. Тестирование включает unit-тесты и property-based тесты для валидационной логики.

## Примечания

- Задачи, отмеченные `*`, являются опциональными и могут быть пропущены для более быстрой реализации MVP
- Каждая задача ссылается на конкретные требования для прослеживаемости
- Контрольные точки обеспечивают инкрементную валидацию
- Property-тесты валидируют универсальные свойства корректности
- Unit-тесты валидируют конкретные примеры и граничные случаи
- Реализация использует JavaScript как для клиентской, так и для серверной части
