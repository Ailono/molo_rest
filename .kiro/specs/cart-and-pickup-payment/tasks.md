# План реализации: Корзина + самовывоз/доставка + оплата через Точка банк

## Обзор

Реализация полного цикла онлайн-заказа с опциями самовывоза и доставки курьером: feature flag → корзина в меню → модал корзины → форма оформления с выбором способа получения → API создания заказа → оплата через Точка банк → фискализация через облачную онлайн-кассу → Telegram-уведомление → админ-панель для управления заказами.

Стек: Node.js + Express, PostgreSQL (Neon), чистый HTML/CSS/JS без фреймворков.

## Примечания

- Задачи, помеченные `*`, являются необязательными и могут быть пропущены для ускорения MVP
- Каждая задача ссылается на конкретные требования для трассируемости
- Property-тесты используют [fast-check](https://github.com/dubzzz/fast-check), unit-тесты — [Vitest](https://vitest.dev/)
- Каждый property-тест помечается комментарием: `// Feature: cart-and-pickup-payment, Property N: <текст>`
- `cart.js` подключается в `menu.html` **до** `menu.js`
- Для работы с Точка банк требуется API ключ, который нужно получить в личном кабинете банка
- Для работы с облачной онлайн-кассой требуется интеграция через API поставщика кассового обслуживания
- Telegram бот должен быть создан через @BotFather и добавлен в чат ресторана
- Для работы с Max (мессенджер от VK) требуется VK API токен и ID чата
- Существующая админ-панель будет расширена настройками уведомлений
- Все чувствительные данные (API ключи, токены) должны храниться в переменных окружения
