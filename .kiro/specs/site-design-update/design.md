# Дизайн-документ: Обновление дизайна и контента сайта

## Overview

Данный документ описывает дизайн-решение для обновления визуального стиля и контента веб-сайта ресторана Molo. Изменения включают замену логотипа, обновление цветовой схемы на тёплые коричневые оттенки (мокко), изменение шрифтов, обновление контента и добавление новых элементов навигации.

### Цели проекта

1. Обновить визуальный идентификатор сайта, заменив текущий логотип на новый
2. Создать тёплую уютную атмосферу с помощью цветовой схемы мокко
3. Использовать фирменный шрифт Novoposelensky для создания уникального стиля
4. Упростить макет главной страницы, удалив декоративные элементы
5. Расширить навигацию с новой страницей "О нас"
6. Убрать лишние заголовки со страницы меню

### Scope

Изменения затрагивают:
- CSS стили (public/css/style.css)
- Главную страницу (public/index.html)
- Страницу меню (public/menu.html)
- Страницу контактов (public/contacts.html)
- Страницу доставки (public/delivery.html)
- Новую страницу about.html

## Architecture

### Текущая архитектура

Сайт построен по простой архитектуре:
- **Frontend**: Статический HTML + CSS + JavaScript
- **No framework**:Vanilla JS, без React/Vue
- **Shared CSS**: Единый файл style.css для всех страниц
- **Header/Footer**: Повторяются на каждой странице (дублирование, а не компоненты)

Архитектура не меняется. Изменения носят визуальный характер:

## Components and Interfaces

### 1. Логотип сайта (site-logo)

**Изменение**: Замена `src="/images/лого.jpg"` на `src="/images/new-logo.png"`

```html
<!-- Текущее -->
<img id="site-logo" src="/images/лого.jpg" alt="Molo" />

<!-- Новое -->
<img id="site-logo" src="/images/new-logo.png" alt="Molo" />
```

**Требования**:
- Атрибут `alt="Molo"` сохраняется для доступности
- Размеры определяются CSS: `height: 48px`

### 2. Навигация

**Добавление вкладки "О нас"**

Текущая навигация:
```html
<nav class="nav">
  <a href="/menu.html">Меню</a>
  <a href="/delivery.html" class="nav-delivery">Заказать доставку</a>
  <a href="/contacts.html">Контакты</a>
</nav>
```

Новая навигация:
```html
<nav class="nav">
  <a href="/menu.html">Меню</a>
  <a href="/about.html">О нас</a>
  <a href="/delivery.html" class="nav-delivery">Заказать доставку</a>
  <a href="/contacts.html">Контакты</a>
</nav>
```

**Требования**:
- Ссылка "О нас" размещается между "Меню" и "Заказать доставку"
- href="/about.html"

### 3. Hero-секция

**Удаление hero-logo-wrap**

```html
<!-- Удалить этот блок -->
<div class="hero-logo-wrap">
  <img src="/images/шапка-обложка-3.png" alt="Molo бистро" />
</div>
```

**Изменение сетки hero-grid**

```css
/* Текущее */
.hero-grid {
  display: grid;
  grid-template-columns: 1fr 420px;
  gap: 28px;
}

/* Новое - одна колонка */
.hero-grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 0;
}
```

**Требования**:
- Полностью удалить элемент `hero-logo-wrap` из HTML
- Адаптировать CSS grid под один столбец
- Сохранить текстовый контент (headline) без изменений

### 4. Время работы

**Изменение на главной странице (hero-meta)**

```html
<!-- Текущее -->
<span>🕐 Ежедневно 10:00 – 22:00</span>

<!-- Новое -->
<span>🕐 Ежедневно 11:00 – 23:00</span>
```

**Изменение в секции "Где мы"**

```html
<!-- Текущее -->
<p class="subtle">Москва, Причальный проезд, 10к1 — ежедневно с 10:00 до 22:00</p>

<!-- Новое -->
<p class="subtle">Москва, Причальный проезд, 10к1 — ежедневно с 11:00 до 23:00</p>
```

**Изменение на странице контактов**

```html
<!-- Текущее -->
<div class="value">Ежедневно с 10:00 до 22:00</div>

<!-- Новое -->
<div class="value">Ежедневно с 11:00 до 23:00</div>
```

**Требования**:
- Формат: "Ежедневно 11:00 – 23:00"
- Обновить на всех страницах (index.html, contacts.html)

### 5. Шрифт Novoposelensky

**Добавление @font-face в CSS**

```css
@font-face {
  font-family: 'Novoposelensky';
  src: url('/font/Novoposelensky.otf') format('opentype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

**Применение шрифта**

```css
/* Текущее */
body {
  font-family: ui-sans-serif, system-ui, -apple-system, ...;
}

/* Новое */
body {
  font-family: 'Novoposelensky', ui-sans-serif, system-ui, -apple-system, ...;
}
```

**Требования**:
- Шрифт загружается из `font/Novoposelensky.otf`
- Применяется ко всему тексту сайта (body)
- Заголовки (h1, h2, h3) сохраняют `Playfair Display`

### 6. Страница меню

**Удаление заголовка "Выберите категорию"**

```html
<!-- Удалить -->
<h1 style="margin:10px 0 14px; font-size: clamp(28px, 3.6vw, 44px); letter-spacing:-.02em;">Выберите категорию</h1>

<!-- Сохранить -->
<div class="kicker">Меню</div>
```

**Требования**:
- Удалить h1 с текстом "Выберите категорию"
- Сохранить элемент kicker с текстом "Меню"
- Функциональность табов категорий не меняется

## Correctness Properties

*Этот раздел не применим к данному проекту.*

**Почему PBT не применяется:**

Данный проект представляет собой обновление визуального стиля (CSS/HTML), а не функциональную логическую систему. Property-based testing не подходит для:

- Изменения CSS переменных (цвета, шрифты)
- HTML шаблонов и разметки
- UI компонентов и их визуального отображения
- Статического контента (текст, изображения)

Для верификации этих изменений используются:
- Визуальное тестирование (ручная проверка)
- Snapshot тесты при необходимости
- Линтинг HTML/CSS

## Error Handling

### Потенциальные ошибки

1. **Шрифт не загружается**
   - Симптом: Текст отображается системным шрифтом
   - Решение: Проверить путь к файлу /font/Novoposelensky.otf, добавить fallback шрифты

2. **Логотип не отображается**
   - Симптом: Пустое место вместо изображения
   - Решение: Проверить путь /images/new-logo.png, убедиться что файл существует

3. **Цвета не применяются**
   - Симптом: Старые цвета остались
   - Решение: Очистить кэш браузера, проверить CSS переменные

4. **Навигация не работает**
   - Симптом: Страница about.html не открывается
   - Решение: Убедиться что файл создан, проверить href

### Проверки перед деплоем

1. Все страницы загружаются без ошибок в консоли
2. Логотип отображается корректно на всех страницах
3. Шрифт Novoposelensky загружается и применяется
4. Цветовая схема мокко применена ко всем элементам
5. Время работы обновлено на всех страницах
6. Навигация содержит ссылку "О нас"
7. Страница about.html доступна по адресу /about.html
8. Заголовок "Выберите категорию" удалён со страницы меню
9. Hero-секция отображается корректно без hero-logo-wrap
10. Адаптивная вёрстка работает на мобильных устройствах
